# React Counter App

A simple Counter Application built with React. This was my first React project deployed live using Netlify + GitHub.

## Live Demo
You can see my project by this link https://lighthearted-croissant-e7e1b0.netlify.app/

## Features
- Increment `+` and Decrement `-` buttons
- Clean, responsive UI
- Count resets on refresh
- Deployed automatically from GitHub
